# Greedy Token Pruning

This repository contains the code for **Greedy Pruning**, a likelihood-preserving method for analyzing and compressing reasoning chains in large language models.

The code supports reproducing the main experiments from the paper, including greedy pruning, baselines, and teacher–pruner–student distillation.

## 📄 Paper

**Do LLMs Encode Functional Importance of Reasoning Tokens?**  

---

## 🧠 Method Overview

Given a question–reasoning–answer triple generated by an LLM, greedy pruning iteratively removes reasoning tokens whose deletion minimally affects model likelihood.  
This induces a token-level importance ranking that reflects the model’s internal notion of functional importance for answer generation.

We evaluate pruned reasoning chains via teacher–pruner–student distillation and compare against multiple token-level pruning baselines.
